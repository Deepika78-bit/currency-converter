# Currency Converter

Hi! This is a currency converter web application developed using JavaScript to perform dynamic currency conversions.
The project focuses on implementing logic and interactivity along with a clean and simple user interface.

# What it does
- Converts one currency to another instantly  
- Handles user input dynamically  
- Simple and interactive interface  

# Built with
- HTML  
- CSS  
- JavaScript  

# Purpose
This project was created to strengthen my understanding of JavaScript concepts and to build an interactive web application.
